# Ralph-x3 Skill

快速启动 Ralph Loop 并运行 3 次迭代的简化命令。

## 快速开始

```
/ralph-x3 "你的任务描述"
```

## 特点

- ✅ **简单** - 一条命令完成，无需记住复杂参数
- ✅ **自动** - 自动设置为 3 次迭代
- ✅ **高效** - 3 次迭代足够处理大部分任务
- ✅ **可预测** - 明确知道会运行几次

## 使用示例

### 代码优化
```
/ralph-x3 "优化 index.html 中的 JavaScript 代码性能"
```

### 添加功能
```
/ralph-x3 "为所有按钮添加 loading 状态和错误处理"
```

### 重构代码
```
/ralph-x3 "重构 config.js，使用更好的命名和组织结构"
```

### 修复 Bug
```
/ralph-x3 "修复导航栏在移动端显示不正确的问题"
```

## 工作流程

```
第 1 次迭代：理解任务并开始工作
    ↓
第 2 次迭代：看到第 1 次的结果并改进
    ↓
第 3 次迭代：看到第 2 次的结果并完善
    ↓
自动停止 或 提前输出 <promise>DONE</promise>
```

## 与完整命令的对比

**使用这个 skill（推荐）：**
```
/ralph-x3 "优化代码"
```

**等效于完整命令：**
```
/ralph-loop "优化代码" --max-iterations 3 --completion-promise "DONE"
```

## 提前结束

如果任务在 3 次迭代前就完成了，输出：

```
<promise>DONE</promise>
```

循环会立即停止，无需等待剩余迭代。

## 取消循环

随时可以取消：

```
/cancel-ralph
```

## 注意事项

1. **描述要具体** - 任务描述越清晰，效果越好
2. **文件要可编辑** - 确保你有权限修改相关文件
3. **目标要明确** - 清楚知道想要达成什么结果
4. **查看每次结果** - 每次迭代都会看到上一次的工作

## 更多迭代？

如果需要更多迭代次数或自定义完成条件，可以直接使用完整命令：

```
/ralph-loop "你的任务" --max-iterations 5 --completion-promise "自定义"
```

## 关键参数说明

- `--max-iterations 3` - 最多运行 3 次迭代
- `--completion-promise "DONE"` - 当检测到 `<promise>DONE</promise>` 时提前结束
